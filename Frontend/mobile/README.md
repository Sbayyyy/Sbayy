## Getting Started

### Prerequisites

- Node.js 18+
- Expo CLI
- iOS Simulator (for iOS development)
- Android Emulator or device (for Android development)

### Installation

```bash
npm install
```

### Development

```bash
# Start development server
npm run dev

# Run on iOS
npm run ios

# Run on Android
npm run android

# Run on web
npm run web
```

## Building

### iOS

```bash
expo build:ios
```

### Android

```bash
expo build:android
```

## Configuration

Edit `app.json` to configure:
- App name and slug
- Icons and splash screens
- Bundle identifiers
- API URL
